# Back to the instance

Back to the instanceはVR chatを起動する際に、前回VR chatをプレイした際に自分がいたインスタンスにホームワールドに移動することなく直接戻ることを可能とするソフトウェアです。GUI版とCLI番があります。
<h2>機能紹介</h2>
<h3>CLI版</h3>
自動的に最後に訪れたインスタンスに戻ります。<br>
<h3>GUI版</h3>
これまでに訪れたインスタンスの一覧を表示します。行きたいものを選んだら、Runかself inviteを選択します。<br>
RunはVRchatをそのインスタントを起動します。Self inviteは自分自身に対してそのインスタントへのインバイトを送ります。<br>
Self inviteの利用にはVR Chatのユーザー名とパスワードが必要です。<br>
<h2>こんな時に使える</h2>
・VR chatが急にクラッシュしてしまい、再度同じインスタンスに戻りたいとき<br>
・これまでに訪れたワールドの確認がしたいとき<br>
・など
<h2>メカニズム</h2>
VR chatのログファイルから前回自分がいたワールド・インスタントのIDを読み取り、VR chatのランチャーにそのインスタントのIDを渡しています。
Self inviteにはVR ChatのAPIを利用しています。そのためVR Chatのユーザー名とパスワードが必要です。<br>
VRChat APIに関しては詳しくは公式サイトを確認してください。
[VRChat API Documentation](https://vrchatapi.github.io/)
